# NovelSmith — Codex 启动指令

项目: https://github.com/liqinyu167/NovelSmith

## 一句话指令

在 `backend/` 下搭建 FastAPI + SQLite 内核：JWT 邀请码认证 + AI Provider 管理（可配置多模型）+ 项目/章节/卡片 CRUD + 写作引擎（大纲/正文生成/润色/扩写）。前端在 `frontend/` 下搭 Vue 3 + CodeMirror 6 编辑器，UI 风格深色主题 + 紫色强调。先跑通本地内核（不依赖 AI 也能做 CRUD），再接入 AI 能力。详细架构见 `ARCHITECTURE.md`，阶段划分见 `ROADMAP.md`，任务拆解见 `docs/build-checklist.md`。
